# media-concatenate
Lets you create one media file from several files.

# Prerequisites
ffmpeg

# Run
Create a mylist.txt by running 
```
  ./generate_input_files.sh
```

and then create output file by running

```
./concatenate.sh
```

Result can be found in output.mp3.
